# E-Portfolio Website Structure

## Site Map

### 1. index.html - Home / About Me
- Hero section with typewriter effect
- Professional summary
- Key achievements
- Call-to-action buttons

### 2. skills.html - Skills & Expertise
- Technical skills with progress bars
- Soft skills visualization
- Tools and technologies radar chart
- Skill categories with icons

### 3. projects.html - Projects Portfolio
- Project cards with hover effects
- Filter by technology/category
- Detailed project descriptions
- GitHub links and live demos

### 4. experience.html - Professional Experience
- Timeline layout
- Student projects and leadership roles
- Achievements and impact metrics
- Skills gained from each experience

### 5. education.html - Education Timeline
- Academic institutions
- Relevant coursework
- Academic achievements
- Expected graduation details

### 6. certificates.html - Certificates & Awards
- Certificate gallery
- Achievement badges
- Skills validation
- Professional development

### 7. contact.html - Contact & Connect
- Contact form
- Social media links
- Professional profiles
- Location and availability

## Navigation Structure
- Fixed navigation bar with smooth scrolling
- Mobile-responsive hamburger menu
- Active page highlighting
- Professional logo/name branding

## Interactive Components
1. Skills radar chart (Chart.js)
2. Project filter and search
3. Contact form validation
4. Timeline animations (Anime.js)
5. Certificate gallery (Splide.js)
6. Progress bars animation
7. Typewriter effects (Typed.js)

## Design Features
- Modern editorial style
- Consistent color palette
- Professional typography
- Smooth animations
- Mobile-first responsive design
- Particle background effects
- Scroll reveal animations
